
# Story Topics — Toastmasters Table Topics App

A fun, interactive React app built for **Tarento India Toastmasters Club (TITMC)** 
to make Table Topics (impromptu speaking) sessions more engaging and 
challenging.

🌐 **Used live in club meetings inside and outside TITMC**

---

## 💡 The Idea

Table Topics is a Toastmasters exercise where members speak 
impromptu for 1–2 minutes on a given topic. I wanted to make it 
more creative and unpredictable — so I built this app as a 
surprise for the club. 

**The twist:** Instead of one topic, words appear one by one as a 
ladder while the speaker talks. They must weave *all* the words 
into a single coherent story — getting harder (and funnier) as 
each new word appears mid-sentence.

---

## 🎮 How It Works

1. A selected speaker starts their story with the **first word**
2. The TT Master clicks to reveal the **next word** — the speaker 
   must include it in their ongoing story
3. Words keep appearing (up to **6 total**) — increasingly 
   unrelated, making it hilariously challenging
4. Automatic timing cards follow Toastmasters rules:
   - 🟢 **Green card** at 1:00 — keep going
   - 🟡 **Yellow card** at 1:30 — start wrapping up  
   - 🔴 **Red card** at 2:00 — time's up

---

## ✨ Features

- Sequential word reveal on TT Master's click
- Auto-timed green / yellow / red card display
- Single page, distraction-free UI for club use
- No login or setup needed — just open and run
